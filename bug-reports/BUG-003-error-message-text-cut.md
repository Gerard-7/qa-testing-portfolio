# BUG-003 - Error message text is cut when message is too long

Application: https://www.saucedemo.com/  
Environment: Desktop / Chrome  
Type: UI / UX  
Severity: Low  

---

## Summary
When the error message is long, part of the text is cut and not fully visible.

---

## Steps to Reproduce
1. Open the login page
2. Trigger an error message (for example, with empty fields)
3. Observe the error message text

---

## Actual Result
The error message is cut and not fully readable.

---

## Expected Result
The full error message should be displayed without being cut.

---

## Impact
Users may not clearly understand the error message.
